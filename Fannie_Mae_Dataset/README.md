
Prediccion de prestamos

----------------------------------------------


Pequeño proyecto que busca predecir si un prestamo llegara a la ejecucion hipotecaria. Fannie Mae compra prestamos de prestamistas

para hacer que estos hagan mas prestamos. La informacion sobre los prestamos y sus resultados lo podemos encontrar [aqui](http://www.fanniemae.com/portal/funding-the-market/data/loan-performance-data.html).
